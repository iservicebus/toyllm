# toyLLM - Demystify Large Language Model (LLM)


## Quick Start


1. In a python env (e.g. conda)

    ```bash
    git clone https://github.com/iservicebus/toyllm.git
    ```

2. In the top level directory run:
    ```bash
    pip install --user  .
    ```
or

    ```bash
    pip install -r requirements.txt    
    ```
